# Company Overview: KhanalTech Solutions

## Purpose

KhanalTech Solutions is a fictional small business used to practise real-world IT support and systems administration.

The goal is to build the kind of environment a junior systems administrator or IT support technician would commonly support.

## Business Profile

- Company size: 25 employees
- Location: One office
- Devices: Windows laptops and desktops
- Server: One Windows Server domain controller
- Identity: Active Directory
- File storage: Shared folders
- Support model: Small internal IT team

## Departments

- Management
- IT
- HR
- Finance
- Sales
- Marketing

## Core Software

- Windows 10 or Windows 11
- Windows Server 2022
- Microsoft Outlook
- Microsoft Teams
- Microsoft Word
- Microsoft Excel
- Google Chrome
- Adobe Reader

## Shared Folders

- Finance
- HR
- Sales
- Marketing
- IT
- Management

## Access Rules

Access should be assigned through groups, not directly to individual users.

Example:

```text
Finance users -> GG_Finance -> Finance shared folder
HR users      -> GG_HR      -> HR shared folder
IT users      -> GG_IT      -> IT shared folder
```

## IT Support Scenarios

The lab will practise these common business support problems:

- Password reset
- Account lockout
- Shared folder access issue
- New user onboarding
- Employee offboarding
- Printer troubleshooting
- VPN troubleshooting
- Suspicious email reporting
- Software access request
